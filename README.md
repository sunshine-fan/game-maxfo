### 游戏名称
maxfo（个人游戏的ID）

### 开发工具
编程:idea / 绘图：PS RGBMaker /后期剪辑 PR 剪映 （音乐：Hawaiian Sun（light version））

### 游戏道具
HP(苹果 面包 HP药剂 红色魔法书)  / MP（葡萄 面包 MP药剂）/ 武器 盔甲 头盔 若干 / 金币+商店 / 钥匙 /技能（火属性，冰属性，毒属性）敌人（黑衣人 飞龙 僵尸 BOSS）   

### 游戏说明 ：

移动：W A S D / 攻击：J  U I O  /道具使用  F1 F2 F3 F4 F5

### 玩法说明
收集所有管卡掉落的的🔑,即为胜利。
